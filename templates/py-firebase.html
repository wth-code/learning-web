Python Firebase Realtime Database

23/2/2021

Install moduels: 

            
pip install firebase
pip install pycryptodome
pip install python_jwt
pip install six
pip install gcloud
pip install sseclient
pip install cryptography
pip install requests-toolbelt

Import moduels: 

                
from firebase import firebase

Setup firebase link: 

                
firebase = firebase.FirebaseApplication("https.......", None)

Post data (insert): 

                    
firebase.post(location, data)

Get data: 

                    
firebase.get(location, tag)  # tag can be blank if want to get all data

Put data (overwirte): 

                    
firebase.put(location, tag, new_data)

Delete data: 

                    
firebase.delete(location, tag)
