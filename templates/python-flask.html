Flask

23/2/2021

Setup

First step, flask's structures. 

            
from flask import *

app = Flask(__name__)

if __name__ == "__main":
    app.run

Flash Messages


first you need to add a secret key: 

            
app.secret_key = b'_5#y2L"F4Q8z\n\xec]/'

Then flash your messages: 

            
flash("Here is your message")

Then add it to your HTML: 


{% with messages = get_flashed_messages() %}
{% if messages %}
{% for message in messages %}
    {{ message }}
{% endfor %}
{% endif %}
{% endwith %}
